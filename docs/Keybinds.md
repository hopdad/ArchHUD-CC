# Keybinds

Complete keybind reference for ArchHUD. For detailed descriptions of each flight feature, see [Flight Controls](Flight-Controls.md).

---

## Flight Controls

| Key | Action | Description |
|-----|--------|-------------|
| W | Pitch Down | Pitch the nose down while held |
| S | Pitch Up | Pitch the nose up while held |
| Q | Roll Left | Roll left while held |
| E | Roll Right | Roll right while held |
| A | Yaw Left | Yaw left while held |
| D | Yaw Right | Yaw right while held |
| Space | Up | Vertical engines up; also initiates takeoff from ground |
| C | Down | Vertical engines down |
| B | Rockets | Toggle rocket engines on/off |
| CTRL | Brake | Toggle brake on/off (in brake toggle mode) or hold to brake (vanilla mode) |

## Throttle and Speed

| Key | Action | Description |
|-----|--------|-------------|
| Scroll Up / R | Speed Up | Raise throttle or cruise speed |
| Scroll Down / T | Speed Down | Lower throttle or cruise speed |
| MMB | Throttle Toggle | If throttle is at 0%, set to 100%; if above 0%, set to 0% |
| Double MMB | Clear All AP | Clears all autopilot features and sets engines to 0 |
| Alt-Scroll Up | Raise Limit | Raise atmosphere speed limit or MaxGameVelocity |
| Alt-Scroll Down | Lower Limit | Lower atmosphere speed limit or MaxGameVelocity |

## Landing and Takeoff

| Key | Action | Description |
|-----|--------|-------------|
| G | Gear/Landing | Brake land from flight, or take off from ground to TargetHoverHeight |
| Alt-6 | Alt Hold / Orbit | Altitude hold in atmosphere; establish orbit in space |
| Alt-6-6 | High Alt / Orbit | Altitude hold at 11% atmosphere height; low orbit height in space |

## Altitude Adjustment

| Key | Action | Description |
|-----|--------|-------------|
| Alt-Space | Raise Altitude | Raise target hover, AGG, or altitude hold height |
| Alt-Shift-Space | Step Raise | Step raise altitude (surface+100m, 11% atmo, low orbit, high orbit) |
| Alt-C | Lower Altitude | Lower target hover, AGG, or altitude hold height |
| Alt-Shift-C | Step Lower | Step lower altitude |

## Autopilot and Navigation

| Key | Action | Description |
|-----|--------|-------------|
| Alt-1 | IPH Forward | Cycle interplanetary helper destination forward |
| Alt-2 | IPH Backward | Cycle interplanetary helper destination backward |
| Alt-4 | Autopilot | Toggle autopilot to selected IPH destination |
| Alt-4-4 | Orbital Hop | Autopilot via low orbit hop (same planet only) |
| Alt-5 | Lock Align | Lock alignment to current IPH target (not in AP) |
| Alt-5-5 | Lock Retro | Lock retrograde (180-degree) alignment |
| Alt-Shift-5 | Lock Pitch | Lock pitch at current value |
| Alt-Shift-8 | Add to Route | Add current IPH target to the route planner |

## Maneuvers

| Key | Action | Description |
|-----|--------|-------------|
| Alt-W | Prograde | Toggle prograde alignment in space (not in AP) |
| Alt-S | Reverse / Retro | 180-degree bank turn in atmosphere; retrograde alignment in space; turn-and-burn toggle during AP |
| Alt-Q | Bank Left | Bank turn left in atmosphere |
| Alt-E | Bank Right | Bank turn right in atmosphere |

## Strafing

| Key | Action | Description |
|-----|--------|-------------|
| Alt-A | Strafe Left | Lateral strafe left |
| Alt-D | Strafe Right | Lateral strafe right |

## Systems

| Key | Action | Description |
|-----|--------|-------------|
| Alt-3 | Toggle HUD | Toggle HUD display on/off |
| Alt-7 | Toggle Sounds | Toggle all sounds on/off |
| Alt-8 | Decoupled | Toggle ground stabilization / decoupled mode (underwater flying) |
| Alt-9 | Gyroscope | Toggle gyroscope on/off |
| Alt-Shift-9 | Engine Tags | Cycle extra engine tags: Off, All, Longitude, Lateral, Vertical |
| Alt-G | AGG | Toggle antigravity generator on/off |
| Alt-B | Transponder | Toggle transponder on/off |
| Alt-CTRL | Handbrake | Lock brakes on (works even when BrakeToggleDefault is off) |

## Defense

| Key | Action | Description |
|-----|--------|-------------|
| Alt-Shift-6 | Vent Shield | Vent shield if available |
| Alt-Shift-7 | Toggle Shield | Toggle shield on/off |

## Utility

| Key | Action | Description |
|-----|--------|-------------|
| Shift-L | Lights | Toggle lights on/off |
| Alt-Shift-L | Freeze Player | Freeze or unfreeze player on remote controller |
| Alt-Shift-1 | Passengers | Show list of boarded players |
| Alt-Shift-2 | Deboard | Force deboard all passengers |
| Alt-Shift-3 | Docked Ships | Show names of docked ships |
| Alt-Shift-4 | Undock | Undock all docked ships |
| LMB | Buttons | Show or hide the button panel; click on-screen buttons |
| Shift | Modifier | Modifier key; also provides remote button access |

## Double-Tap Shortcuts

Some keys support a rapid double-tap for alternate actions:

| Key | Action | Description |
|-----|--------|-------------|
| Alt-4-4 | Orbital Hop | Autopilot via low orbit hop instead of direct route |
| Alt-5-5 | Lock Retro | Lock retrograde alignment instead of prograde |
| Alt-6-6 | High Alt / Orbit | Altitude hold at 11% atmo or low orbit height |
| Double MMB | Clear All AP | Full autopilot reset and engine cutoff |

---

## ALT-Mousewheel Shortcuts

| Key | Action | Description |
|-----|--------|-------------|
| Alt-Scroll Up | Raise Limit | Increase atmosphere speed limit (up to AtmoSpeedLimit) or MaxGameVelocity |
| Alt-Scroll Down | Lower Limit | Decrease atmosphere speed limit or MaxGameVelocity |
| Alt-MMB | Toggle Limit | Toggle speed limit between 0 and AtmoSpeedLimit |
